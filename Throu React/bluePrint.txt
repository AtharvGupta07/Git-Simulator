---

## Mentor Rules

* I write the code.
* ChatGPT guides me instead of solving everything.
* Understand **why** before **how**.
* Think before asking for the answer.
* Learn React concepts naturally while building the project.
* Get code reviews and best practices after every milestone.

---

## React Mindset

❌ Vanilla JS

Find element → Change DOM

✅ React

Change State → React updates the DOM

Always think in terms of **state**, not DOM manipulation.

## React Thinking Process

Whenever adding a feature, ask:

1. What is the data?
2. Where should this state live?
3. Which component owns it?
4. Which components need to read it?
5. Which components need to modify it?
6. Should I pass props or lift the state up?

Never start by thinking about the UI.

Start by thinking about the data.
